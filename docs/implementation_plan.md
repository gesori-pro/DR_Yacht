# Yacht Dice 4인 실시간 게임 구현 계획서

## 개요

닌텐도 스위치 "세계 게임전집 51" 버전의 Yacht Dice를 기반으로 한 4인 실시간 온라인 게임.
Firebase Realtime Database를 활용한 실시간 멀티플레이어 지원.

---

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프론트엔드 | HTML5, CSS3, JavaScript (Vanilla) |
| 실시간 동기화 | Firebase Realtime Database |
| 인증 | Firebase Anonymous Authentication |
| 호스팅 | GitHub Pages, Netlify |
| 폰트 | Google Fonts (Jua) |

---

## 게임 룰 (닌텐도 스위치 버전)

### 기본 규칙
- 5개의 주사위 사용
- 턴당 최대 3번 굴리기
- 45초 턴 제한

### 점수표

#### 상단 섹션
| 카테고리 | 점수 |
|----------|------|
| Ones ~ Sixes | 해당 숫자 합계 |
| **보너스** | 상단 합계 63점 이상 시 +35점 |

#### 하단 섹션
| 카테고리 | 조건 | 점수 |
|----------|------|------|
| Three of a Kind | 동일 숫자 3개+ | 주사위 합계 |
| Four of a Kind | 동일 숫자 4개+ | 주사위 합계 |
| Full House | 3+2 조합 | **주사위 합계** |
| Small Straight | 연속 4개 | **15점** |
| Large Straight | 연속 5개 | **30점** |
| Chance | 없음 | 주사위 합계 |
| Yacht | 동일 숫자 5개 | **50점** |

## 기능 상세

### 1. 로비 화면
- 닉네임 입력 (한글 10자 제한)
- 방 생성 버튼
- 방 코드 입력 + 참가 버튼
- 랜덤 매칭 버튼
- 다크 모드 토글

### 2. 대기실 화면
- 방 코드 표시 (복사 기능)
- 플레이어 목록 (최대 4명)
- 방장 표시 (왕관 아이콘)
- 대기 시간 카운트다운 (2분)
- 게임 시작 버튼 (방장 전용, 2인 이상 시 활성화)
- 나가기 버튼

### 3. 게임 화면

#### 상단 플레이어 바
- 4명의 플레이어 정보
- 현재 턴 플레이어 강조
- 각 플레이어 현재 총점

#### 중앙 주사위 영역
- 5개의 주사위 (둥근 모서리)
- 주사위 선택/해제 (킵 기능)
- 굴리기 버튼 (남은 횟수 표시)
- 남은 시간 표시 (45초)
- 3D 회전 애니메이션

#### 하단 스코어보드
- 13개 카테고리
- 예상 점수 미리보기
- 선택 가능한 카테고리 하이라이트
- 완성된 패 강조 (반짝임 → 골드/레인보우)

### 4. 턴 순서 결정 화면
- 룰렛 스타일 애니메이션
- 결정된 순서 표시

### 5. 결과 화면
- 순위표 (1~4위)
- 각 플레이어 최종 점수
- 로비로 돌아가기 버튼
